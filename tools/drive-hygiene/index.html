JAB Drive Hygiene Check

A safe, read-only disk inspection and cleanup utility built on native Windows tooling. Understand your storage before you act.

Free

What This Tool Does

  • Inspects disk health and SMART status across all connected drives
  • Reports TRIM and optimization status for SSDs
  • Provides a clear breakdown of disk usage by category
  • Identifies temporary files, system caches, and other cleanup candidates
  • Optionally invokes native Windows Disk Cleanup with pre-selected safe categories

What This Tool Does Not Do

  • No promises of performance improvements or speed boosts
  • No background services or persistent agents
  • No registry modifications of any kind
  • No driver installations or kernel-level changes
  • No automated destructive actions without explicit confirmation
  • No telemetry, no data collection, no phone-home behavior

Capabilities

Disk Inspection

Read-only analysis of drive health, capacity, and file system state.

Usage Breakdown

Clear categories showing what is consuming space and where.

SSD Health

TRIM status, optimization state, and wear-level indicators for solid-state drives.

Safe Cleanup

Uses native Windows Disk Cleanup with safe, pre-vetted categories only.

Why This Exists

This tool was built during real production support work. When troubleshooting application issues, one of the first questions is always about disk state: Is there space? Is the drive healthy? Are temp files piling up?

Most cleanup utilities do too much. They promise speed boosts, modify registries, and run background agents. We wanted something operators could trust to run on any machine—inspection first, action second, always transparent.
